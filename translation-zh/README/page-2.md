> Appearance`

## Via BRAT

要通过 BRAT 安装此主题, 请导航到`Settings
> Community Plugins
> Browse`
一、导 言.`Type "BRAT" in the search bar
> Select it (the one by TftHacker)
> Click install`
2. 国家 等待安装完成然后单击`Enable`
3个 单击`Options`
4. 点击`Add Beta Theme`
5.五。 投入`https://github.com/bladeacer/flexcyon`并单击`Add Theme`
