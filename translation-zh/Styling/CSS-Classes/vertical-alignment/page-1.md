## Usage
水平应用 在读取模式中左对右( LTR) 到您的笔记
```md
---
cssclasses
    - vertical-ltr
---
```

### Callout metadata provider
垂直 LTR 还有一个点名元数据提供者

```md
>[!info|vertical-ltr] The title will not display
> The contents will be displayed as vertical ltr
```
