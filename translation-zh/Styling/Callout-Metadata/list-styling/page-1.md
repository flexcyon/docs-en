-  上希腊语没有标准执行

"在列表项目和文本一起"是什么意思?
?
> Effectively, the list item number/letter/whatever will inherit the indentation of the list item. Think of writing:
```md
  1. Never
  2. Gonna
  3. Give
```

改为
```
1. Never
2. Gonna
3. Give
```
