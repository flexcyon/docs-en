### Inline title font size

目标CSS变量:`var(--inline-title-size)`

默认: 1.802(em)

### Inline title font weight

目标CSS变量:`var(--inline-title-weight)`

默认: 700

### Inline title font height

目标CSS变量:`var(--inline-title-height)`

默认: 1.2
