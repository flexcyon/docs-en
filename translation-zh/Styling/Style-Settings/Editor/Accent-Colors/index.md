---
title: 调色器
icon: material/format-color-fill
---

定义的颜色覆盖标题颜色和其他UI元素的主题默认值
包括超音调颜色
> 这些颜色超过了标准cssvar(- color- cyan) 等变量

接受格式:RGB,HEX

□ 导航

```md
Style Settings
|-- 。
|-- Flexcyon Style Settings
|   |-- Editor
|   |   |-- Accent Colors
|   |   |-- 。
|   |-- 。
|-- 。
```

□ 配置选项
