## Configuration Options

### Media Embed Vertical margin

目标CSS变量:`var(--flexcyon-media-embed-vertical-margin)`

默认: 8 (px)

### Image border radius

目标CSS变量:`var(--img-border-radius)`

默认: 15 (px)
