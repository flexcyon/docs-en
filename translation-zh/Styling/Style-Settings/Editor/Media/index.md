## Configuration Options

### Media Embed Vertical margin

CSS目标变量:`var(--flexcyon-media-embed-vertical-margin)`
默认: 8 (px)

### Image border radius

CSS目标变量:`var(--img-border-radius)`
默认: 15 (px)
