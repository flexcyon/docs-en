## Configuration Prompts

### Prompt width

。CSS。:`var(--prompt-width)`

Default: 700 (px)

### Prompt max width

。CSS。:`var(--prompt-max-width)`

Default: 80 (vw)

### Prompt max height

。CSS。:`var(--prompt-max-height)`

Default: 70 (vh)
