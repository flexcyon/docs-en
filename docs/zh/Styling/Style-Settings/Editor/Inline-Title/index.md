### 行内标题字体大小

目标 CSS 变量: `var(--inline-title-size)`

默认: 1.802 (em)

### 行内标题字重

目标 CSS 变量: `var(--inline-title-weight)`

默认: 700

### 行内标题行高度

目标 CSS 变量: `var(--inline-title-height)`

默认: 1.2
