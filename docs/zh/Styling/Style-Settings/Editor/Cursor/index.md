## 配置选项

### 启用平滑光标

目标 CSS 类: `var(--flexcyon-enable-smooth-cursor)`

默认: true

### 平滑光标过渡持续时间

目标 CSS 类: `var(--flexcyon-cursor-duration)`

默认: 42.5 (ms)

### 平滑光标计时函数

目标 CSS 类: `var(--flexcyon-cursor-timing-fn)`

默认: ease-in

### 平滑光标最小宽度

目标 CSS 类: `var(--flexcyon-cursor-min-width)`

默认: unset
