## 配置选项

### 媒体嵌入垂直边距

目标 CSS 变量: `var(--flexcyon-media-embed-vertical-margin)`

默认: 8 (px)

### 图像边框半径

目标 CSS 变量: `var(--img-border-radius)`

默认: 15 (px)
