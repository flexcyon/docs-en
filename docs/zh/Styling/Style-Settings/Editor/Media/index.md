## Configuration Options

### Media Embed Vertical margin

CSS Variable(s) targeted: `var(--flexcyon-media-embed-vertical-margin)`

Default: 8 (px)

### Image border radius

CSS Variable(s) targeted: `var(--img-border-radius)`

Default: 15 (px)
