配置图形视图控件。

接受的格式: px

## 配置选项

### 图形视图控件垂直填充

目标 CSS 变量: `var(--graph-control-verti-padding)`

默认: 6 (px)

### 图形视图控件水平填充

目标 CSS 变量: `var(--graph-control-horiz-padding)`

默认: 12 (px)
