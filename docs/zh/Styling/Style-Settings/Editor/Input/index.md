## 配置选项

### 输入边界半径

目标 CSS 变量: `var(--input-radius)`

默认: 0 (px)

### 输入框垂直填充

目标 CSS 变量: `var(--flexcyon-input-verti-padding)`

默认: 4 (px)

### 输入框水平填充

目标 CSS 变量: `var(--flexcyon-input-horiz-padding)`

默认: 8 (px)
