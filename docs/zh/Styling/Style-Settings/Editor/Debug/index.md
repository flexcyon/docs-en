## 配置选项

### 在视图标题中显示设备操作系统详细信息

目标 CSS 类: `.flexcyon-display-os`

默认: false (类切换)

### 设备操作系统详细信息右侧填充

目标 CSS 变量: `var(--flexcyon-os-detail-padding-r)`

默认: 24 (px)

### 显示屏尺寸

目标 CSS 类: `.flexcyon-display-screen-dimens`

默认: false (类切换)

### 显示屏尺寸左侧填充

目标 CSS 变量: `var(--flexcyon-screen-dimens-padding-l)`

默认: 16 (px)
