Callout 元数据: "italic-title"

用法:

```md
> [!info|italic-title] The title will be displayed as italic
> Content is shown as usual
```

这种风格本身也适用于 ["italic-all"](../combined-styling/page-18.md) 和 ["italic-content"](../content-styling/page-8.md)。
