Callout 元数据: "ltr-content"

用法:

```md
> [!info|ltr-content] The title will be displayed as usual
> The content will be displayed as ltr
```

这种风格本身也适用于 ["ltr-all"](../combined-styling/page-12.md) 和 ["ltr-title"](../title-styling/page-12.md)。
