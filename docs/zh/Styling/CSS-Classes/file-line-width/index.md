# 用法

按每个文件配置文件行宽度

```md
cssclasses:
    - editor-50vw
---

Your content will have a file line width of 50 viewport width (50 percent of your
screen's size)
```

支持的其他值:

- `.editor-50vw`
- `.editor-55vw`
- `.editor-60vw`
- `.editor-65vw`
- `.editor-70vw`
- `.editor-75vw`
- `.editor-80vw`
- `.editor-85vw`
- `.editor-90vw`
- `.editor-95vw`
