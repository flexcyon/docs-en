# 用法

在 Reading 模式下对笔记应用水平从左到右（LTR）

```md
---
cssclasses
    - vertical-ltr
---
```

## Callout 元数据提供者

垂直 LTR 还有一个标注元数据提供程序

```md
>[!info|vert-ltr] The title will not display
> The contents will be displayed as vertical ltr
```
