# 用法

在 Reading 模式下对笔记应用水平从右向左（RTL）

```md
---
cssclasses
    - tategaki
---
```

## Callout 元数据提供者

Tategaki 还有一个 Callout 元数据提供者

```md
>[!info|tategaki] The title will not display
> The contents will be displayed as vertical rtl
```
