### Smiley Toggle Icons in Settings

CSS Classe(s) targeted: `.flexcyon-ascii-icon-set`

### Ensure community plugin icons take precedence

CSS Classe(s) targeted: `.flexcyon-ensure-plugin-icon`

Default: true (class toggle)
