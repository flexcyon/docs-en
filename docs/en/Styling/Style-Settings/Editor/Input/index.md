## Configuration Options

### Input border radius

CSS Variable(s) targeted: `var(--input-radius)`

Default: 0 (px)

### Input box vertical padding

CSS Variable(s) targeted: `var(--flexcyon-input-verti-padding)`

Default: 4 (px)

### Input box horizontal padding

CSS Variable(s) targeted: `var(--flexcyon-input-horiz-padding)`

Default: 8 (px)
