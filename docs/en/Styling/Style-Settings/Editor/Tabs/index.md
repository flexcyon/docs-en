## Configuration Options

### Number tabs

Enabled by default in RTZ mode.

CSS Classe(s) targeted: `.flexcyon-number-tabs`

Default: false (class toggle)

### Remove Tab Headers

Enabled by default in RTZ mode.

CSS Classe(s) targeted: `.flexcyon-remove-tab-headers`

Default: false (class toggle)
