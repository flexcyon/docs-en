## Configuration Options

### View header breadcrumb max width

CSS Variable(s) targeted: `var(--flexcyon-view-header-breadcrumb-max-w)`

Default: 12.5 (vw)

### Mode View Header Title Vertical Padding

Vertical padding of View header title in modes like Angled and Powerlevel10k.

CSS Variable(s) targeted: `var(--flexcyon-mode-view-header-title-verti-padding)`

Default: 4 (px)

### Mode View Header Title Horizontal Padding

Horizontal padding of View header title in modes like Angled and Powerlevel10k.

CSS Variable(s) targeted: `var(--flexcyon-mode-view-header-title-horiz-padding)`

Default: 12 (px)
