> `ol-style_type`

```md
>[!info|ol-lower-greek] 평소와 같은 제목
> 1. 숫자 1은 읽기/실시간 미리보기에서 그리스 문자 알파로 표시됩니다.
> 2. 숫자 2는 읽기/실시간 미리보기에서 그리스 문자 베타로 표시됩니다.
> 3. 숫자 3은 읽기/실시간 미리보기에서 그리스 문자 감마로 표시됩니다.
> ...
```

___

- 상위 그리스 문자에 대한 표준 구현은 없습니다.

목록 항목 내부와 텍스트가 함께한다는 것은 무엇을 의미하나요?

> 실제로, 목록 항목의 번호/글자/그 어떤 것이든 해당 목록 항목의 들여쓰기를 상속하게 됩니다.
> 다음과 같이 작성한다고 생각해 보세요:

```md
  1. Never
  2. Gonna
  3. Give
```

대신

```
1. Never
2. Gonna
3. Give
```
