---
title: 글꼴 크기
---

커스텀 콜아웃 / 콜아웃 메타데이터:

- "title-`<불투명도값>`"

- "content-`<불투명도값>`"

- "all-`<불투명도값>`"

옵션: Smaller, Small, Medium, Large

[UI 글꼴 크기](../../../style-settings/editor/ui-font-size) 설정에서 구성하세요.

사용법:

```md
> [!info|title-smaller content-medium] 콜아웃 제목은 작은 UI 글꼴 크기를 상속받습니다.
> 콜아웃 내용은 중간 UI 글꼴 크기를 상속받습니다.
```

```md
> [!info|all-small] 콜아웃 제목은 작은 UI 글꼴 크기를 상속받습니다.
> 콜아웃 내용은 작은 UI 글꼴 크기를 상속받습니다.
```
