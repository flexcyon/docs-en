# 사용법

노트별로 파일 줄 너비를 구성하세요.

```md
cssclasses:
    - editor-50vw
---

콘텐츠는 파일 줄 너비가 뷰포트 너비의 50% (화면 크기의 50퍼센트)로 설정됩니다.
```

지원되는 다른 값:

- `.editor-50vw`
- `.editor-55vw`
- `.editor-60vw`
- `.editor-65vw`
- `.editor-70vw`
- `.editor-75vw`
- `.editor-80vw`
- `.editor-85vw`
- `.editor-90vw`
- `.editor-95vw`
