# 사용법

편집기에 텍스트 들여쓰기와 줄 간격을 늘려줍니다. 크기는 Style Settings에서 조정할 수 있습니다.

```md
cssclasses:
    - editor-writing
---

콘텐츠는 텍스트 들여쓰기와 줄 간격이 늘어납니다.
```

<!-- Style Settings 조정 링크 -->

## 콜아웃 메타데이터 제공자

Writing 모드용 콜아웃 메타데이터 제공자도 있습니다(라이브 미리보기 및 읽기 모드에서도 작동합니다).

```md
>[!note|writing] 평소와 같이 표시됩니다
> 텍스트 들여쓰기와 줄 간격이 늘어납니다 
```
