허용되는 형식: px


## 환경 설정 옵션

### 그래프 뷰 컨트롤 세로 패딩

대상 CSS 변수: `var(--graph-control-verti-padding)`

기본값: 6 (px)

### 그래프 뷰 컨트롤 가로 패딩

대상 CSS 변수: `var(--graph-control-horiz-padding)`

기본값: 12 (px)
