---
title: Accent Colors
---

여기서 정의한 색상은 제목을 비롯한 여러 UI 요소의 테마 기본 색상을 덮어씁니다.
Accent Color도 함께 덮어씁니다.

> 이러한 색상은 var(--color-cyan)와 같은 표준 CSS 변수를 덮어씁니다.

허용되는 형식: RGB, HEX

## 내비게이션

```md
Style Settings
|-- ...
|-- Flexcyon Style Settings
| |-- Editor
| | |-- Accent Colors
| | |-- ...
| |-- ...
|-- ...
```

## 환경 설정 옵션

### 시안 색상

대상 CSS 변수: `var(--flexcyon-cyan)`, `var(--color-cyan-rgb)`

기본값(라이트 모드):
<span class="col-sqr" style="background-color: #3b9ba1"></span> #3b9ba1

기본값(다크 모드):
<span class="col-sqr" style="background-color: #3cb9c2"></span> #3cb9c2
