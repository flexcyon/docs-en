## 환경 설정 옵션

### View Header 브레드크럼 최대 너비

대상 CSS 변수: `var(--flexcyon-view-header-breadcrumb-max-w)`

기본값: 12.5 (vw)

### 모드별 View Header 제목 세로 패딩

Angled 모드 및 Powerlevel10k와 같은 모드에서의 View Header 제목 세로 패딩.

대상 CSS 변수: `var(--flexcyon-mode-view-header-title-verti-padding)`

기본값: 4 (px)

### 모드별 View Header 제목 가로 패딩

Angled 모드 및 Powerlevel10k와 같은 모드에서의 View Header 제목 가로 패딩.

대상 CSS 변수: `var(--flexcyon-mode-view-header-title-horiz-padding)`

기본값: 12 (px)
