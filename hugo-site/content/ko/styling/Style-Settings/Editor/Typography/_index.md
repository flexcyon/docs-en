## 환경 설정 옵션

### 무채색 텍스트 색상

대상 CSS 변수: `var(--flexcyon-text-muted)`

기본값:
<span class="col-sqr" style="background-color: #6f768599"></span> #6f768599

### 일반 텍스트 색상

대상 CSS 변수: `var(--text-normal)`

기본값(라이트 모드):
<span class="col-sqr" style="background-color: #080808"></span> #080808

기본값(다크 모드):
<span class="col-sqr" style="background-color: #d3d5d3"></span> #d3d5d3

### 기본 줄 높이

대상 CSS 변수: `var(--line-height-normal)`

기본값: 1.5 (px)

### 촘촘한 줄 높이

대상 CSS 변수: `var(--line-height-tight)`

기본값: 1.3 (px)

### 기본 단어 간격

대상 CSS 변수: `var(--flexcyon-w-spacing)`

기본값: 0 (px)

### 기본 글자 간격

대상 CSS 변수: `var(--flexcyon-l-spacing)`

기본값: 0 (px)

### 강조 텍스트 색상

대상 CSS 변수: `var(--flexcyon-highlight-colour)`

기본값: 시안 (변수 선택)

옵션:

- 시안
- 파랑
- 보라
- 분홍
- 빨강
- 오렌지
- 노랑
- 초록

기본값: 22.5%
