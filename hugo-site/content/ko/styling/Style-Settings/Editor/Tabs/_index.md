## 환경 설정 옵션

### 숫자 탭

RTZ 모드에서 기본적으로 활성화됩니다.

대상 CSS 클래스: `.flexcyon-number-tabs`

기본값: false (클래스 토글)

### 탭 헤더 제거

RTZ 모드에서 기본적으로 활성화됩니다.

대상 CSS 클래스: `.flexcyon-remove-tab-headers`

기본값: false (클래스 토글)
