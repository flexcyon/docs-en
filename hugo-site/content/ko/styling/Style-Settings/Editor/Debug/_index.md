## 환경 설정 옵션

### View Header에 디바이스 운영체제 세부 정보 표시

대상 CSS 클래스: `.flexcyon-display-os`

기본값: false (클래스 토글)

### 디바이스 운영체제 세부 정보 왼쪽 패딩

대상 CSS 변수: `var(--flexcyon-os-detail-padding-l)`

기본값: 12 (px)

### 디바이스 운영체제 세부 정보 오른쪽 패딩

대상 CSS 변수: `var(--flexcyon-os-detail-padding-r)`

기본값: 8 (px)

### 디스플레이 화면 치수 표시

대상 CSS 클래스: `.flexcyon-display-screen-dimens`

기본값: false (클래스 토글)

### 디스플레이 화면 치수 왼쪽 패딩

대상 CSS 변수: `var(--flexcyon-screen-dimens-padding-l)`

기본값: 16 (px)
