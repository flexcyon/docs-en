---
title: 기본 색상
---

정의된 색상은 배경 및 기타 UI 요소에 사용되는 기본 색상에 대한 테마 기본값을 덮어씁니다.

허용되는 형식: HEX

## 내비게이션

```md
Style Settings
|-- ...
|-- Flexcyon Style Settings
|   |-- Editor
|   |   |-- ...
|   |   |-- Base Colors
|   |   |-- ...
|   |-- ...
|-- ...
```

## 환경 설정 옵션

### 기본 색상 01

대상 CSS 변수: `var(--flexcyon-base-01)`

기본값(라이트 모드):
<span class="col-sqr" style="background-color: #faf7ef"></span> #faf7ef

기본값(다크 모드):
<span class="col-sqr" style="background-color: #14161c"></span> #14161c
