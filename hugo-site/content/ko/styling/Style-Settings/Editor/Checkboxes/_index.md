## 환경 설정 옵션

허용되는 형식: rem

### ASCII 체크박스 활성화

대상 CSS 변수: `var(--flexcyon-ascii-checkboxes-enabled)`

기본값: true (클래스 토글)

### ASCII 체크박스 글꼴 크기

대상 CSS 변수: `var(--flexcyon-ascii-checkboxes-font-size)`

기본값: 1.2 (rem)

<span style="font-size: 1.2rem">ASCII 체크박스 글꼴 크기 샘플</span>

### 클립 패스 체크박스 활성화

대상 CSS 변수: `var(--flexcyon-clip-path-checkboxes-enabled)`

기본값: false (클래스 토글)
