## 환경 설정 옵션

### 내부 링크 색상

대상 CSS 변수: `var(--flexcyon-link-color)`

기본값(라이트 모드):
<span class="col-sqr" style="background-color: #6f49ae"></span> #6f49ae


기본값(다크 모드): <span class="col-sqr" style="background-color: #a461c8"></span> #a461c8

### 외부 링크 색상
대상 CSS 변수: `var(--flexcyon-external-link-color)`

기본값(라이트 모드): <span class="col-sqr" style="background-color: #5c9fe4"></span> #5c9fe4

기본값(다크 모드): <span class="col-sqr" style="background-color: #5a8fcd"></span> #5a8fcd

### 미해결 링크 색상
대상 CSS 변수: `var(--flexcyon-link-unresolved-color)`

기본값(라이트 모드): <span class="col-sqr" style="background-color: #6f49ae"></span> #6f49ae

기본값(다크 모드): <span class="col-sqr" style="background-color: #a461c8"></span> #a461c8

### 대체 링크 스타일 활성화

Willemstad 테마의 제목에서 영감을 받았습니다.

대상 CSS 클래스: `var(--flexcyon-alt-link-style)`

기본값: false (클래스 토글)
