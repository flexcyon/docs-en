파일의 속성/메타데이터/YAML에 대한 CSS 변수를 구성합니다.

## 내비게이션

```md
Style Settings
|-- ...
|-- Flexcyon Style Settings
|   |-- Editor
|   |   |-- ...
|   |   |-- Properties
|   |   |-- ...
|   |-- ...
|-- ...
```

## 환경 설정 옵션

허용되는 형식: px

### 읽기 모드에서 속성을 표시하지 않음

대상 CSS 변수: `var(--flexcyon-metadata-display-reading-disabled)`

기본값: false (클래스 토글)

### 라이브 미리보기 모드에서 속성을 표시하지 않음

대상 CSS 변수: `var(--flexcyon-metadata-display-editing-disabled)`

기본값: false (클래스 토글)

### 무지개 메타데이터 아이콘 활성화

대상 CSS 변수: `var(--flexcyon-metadata-rainbow-icons)`

기본값: false (클래스 토글)

### 메타데이터 컨테이너 왼쪽 패딩

대상 CSS 변수: `var(--flexcyon-meta-container-padding-left)`

기본값: 14 (px)

### 속성 제목 표시 안 함

대상 CSS 클래스: `var(--flexcyon-metadata-no-properties-title)`

기본값: false (클래스 토글)
