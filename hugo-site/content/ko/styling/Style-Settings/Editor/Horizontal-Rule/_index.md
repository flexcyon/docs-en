## 환경 설정 옵션

### 읽기 모드에서 가로줄 비활성화
대상 CSS 클래스: `.flexcyon-hr-no-reading`

기본값: false (클래스 토글)

### 가로줄 색상
대상 CSS 변수: `var(--hr-color)`

기본값: #393e48

<span color="#393e48">예시 가로줄 색상</span>

### 가로줄 내부에 커스텀 문자열 추가
대상 CSS 클래스: `.flexcyon-cust-str-hr`

기본값: false (클래스 토글)

### 가로줄 커스텀 문자열 내용
대상 CSS 변수: `var(--flexcyon-cust-hr-str)`

기본값: "f"

### 가로줄 커스텀 문자열 글꼴 크기
대상 CSS 변수: `var(--flexcyon-cust-hr-str-font-size)`

기본값: 1.118 (em)

### 가로줄 커스텀 문자열 가로 패딩
대상 CSS 변수: `var(--flexcyon-cust-hr-str-horiz-padding)`

기본값: 8 (px)
