---
title: 디자인 원칙
---

이 테마는 다음을 달성하고자 합니다.

1. '뚜렷한 방향성'을 갖고, 사용자가 자주 사용하는 기능들을 묶되 일부 특수한 사용 사례도 다룹니다.

2. 적당히 가벼우며, 이는 *임베드를 전혀 사용하지 않는다는 의미*입니다(폰트, SVG 등 포함되지 않음).

3. 합리적인 기본 설정

4. Style Settings를 통해 커스터마이징 가능

5. 비활성화되거나 초점이 맞춰지지 않은 UI 요소를 흐리게 표시하여 정보 과부하를 줄입니다.

6. 수많은 콜아웃 메타데이터 유틸리티와 cssclasses를 하나로 묶습니다.
