Halcyon, Flexoki 및 Origami의 색상 조합을 결합한 Obsidian 테마입니다.

저는 [Halcyon 색상 조합](https://halcyon-theme.netlify.app/)의 생기 넘치는 색감과 [Flexoki 색상 조합](https://stephango.com/flexoki)의 잉크 같은 미학을 정말 좋아했습니다. 그래서 이 두 가지를 결합해 이 테마를 만들게 되었습니다.

### 이 테마가 갖춘 기능

- vim 모드 상태 (내장 vim 키바인드 사용 시)
- Powerlevel10k에서 영감받은 상태바 스타일링과 같은 상태바 옵션
- 스마일리 토글 아이콘 및 기타 설정 옵션
- 새 빈 탭에 ASCII 아트 또는 커스텀 인용문
- UI 요소에 대한 미묘한 불투명도 효과
- 다크 모드와 라이트 모드 색상 조합
- ASCII 체크박스
- 밑줄, 번호 매기기, 색상 지정 등 제목 옵션
- 수많은 [`cssclasses` 및 `callout` 커스텀 옵션](../styling/css-classes)
- TUI에서 영감받은 애드온이 있는 카드 레이아웃, Angled 레이아웃 등의 워크스페이스 레이아웃
- 왼쪽, 오른쪽 사이드바 및 모달 배경 이미지 구성
- 탭에 대한 애니메이션 옵션
- 그 외에도 많은 기능들...

### 이 테마가 갖추지 못한 것

- 여러 색상 조합(다만 기존 색상을 직접 재정의할 수 있습니다. 자세한 내용은 [Style Settings](../styling/style-settings) 참조)
- 대체 체크박스
- 내장 폰트, `svg`
