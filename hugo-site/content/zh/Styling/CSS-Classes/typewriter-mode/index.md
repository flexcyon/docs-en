# 用法

在源代码和实时预览模式下, 降低除活动行之外的所有行的不透明度.
可以在 "样式设置" 中调整非活动线的不透明度.

```md
cssclasses:
    - typewriter-mode
---

Lines before the active line are dimmed.

Active line has full opacity.

Lines after the active line are dimmed.
```
