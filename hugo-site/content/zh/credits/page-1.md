### 字体

- [Cascadia Code](https://github.com/microsoft/cascadia-code) 是一种有趣的新等宽字体
包括编程连字, 旨在增强现代外观和感觉 Windows Terminal.

字体根据 [SIL Open Font License](https://openfontlicense.org) 获得许可.

- [Departure Mono](https://departuremono.com) 是 Helena Zhang 的等宽像素字体.

字体根据 [SIL Open Font License](https://openfontlicense.org) 获得许可.

- [Maple Mono](https://github.com/subframe7536/Maple-font) 是一种开源的等宽字体
IDE 和 Termina 具有圆角, 连字和 Nerd 字体图标，细粒度定制选项.

其中英文完美 2:1 对齐的特点使字体能够在多语言等场景中实现整洁, 统一, 美观, 舒适
的外观显示和 Markdown 表格.

字体根据 [SIL Open Font License](https://openfontlicense.org) 获得许可.
