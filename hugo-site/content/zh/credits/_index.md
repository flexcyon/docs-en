- `@OWA/bennyyip` 在 Obsidian Members Group (OMG) Discord 为 tategaki 写的 CSS

- [Ukiyo 主题](https://github.com/technerium/obsidian-ukiyo) 由 vaykinov 和 wizentex 为弹出 callout

- `@Kapirklaa` 在 OMG Discord (`@ElsaTam` 在 GitHub) 的背景图像 CSS

- `@Nuno` 在 OMG Discord 的标签动画 CSS

- `@LennZone` 的 [Enhanced File Explorer 主题](https://github.com/LennZone/enhanced-file-explorer-tree), 用于标记文件树风格

- `Bluemoondragon07`' 的
[Wikipedia 主题](https://github.com/bluemoondragon07/Wikipedia-Theme)
(在 GitHub 上) 用于他们的 Wikipedia 风格的 infoboxes/asides.

- `efemkay`s [Modular CSS Layout (MCL) 代码段](https://github.com/efemkay/obsidian-modular-css-layout/)
(在 GitHub 上) 用于原始的 multi-column 实现.

- `@kneecaps` 在 Obsidian Members Group (OMG) Discord 为他们的 multi-column 布局 + aside float 代码段.
你可能会从创建 [Origami](https://github.com/7368697661/Origami) 和 [Ultra Lobster](https://github.com/7368697661/Ultra-Lobster) 主题中认得他们.
