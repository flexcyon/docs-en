配置无障碍功能选项.

当前选项允许您为主题的颜色或全局配置亮度, 对比度, 饱和度等.

在达到全局过滤器效果之前配置与主题相关的无障碍功能设置对性能更好.
主题相关设置也动态地显示了主题本身的几乎所有颜色.

## 导航

```md
Style Settings
|-- flexcyon://Accessibility
```

## 配置选项

### 主题色彩对比偏移

定义主题颜色的对比度偏移.
对于暗模式颜色, 较大的正值会增加对比度, 而对于亮模式颜色则相反.

目标 CSS 变量: `var(--flexcyon-theme-contrast-offset)`

默认: 0 (可变滑块, 范围从 -0.3 到 0.3)
> 大于 +/- 0.3 的值对于日常使用来说并不太合理.
> 如果需要更大范围的值, 请告诉我.

### 主题色暖色偏移

定义温暖, 抵消主题的颜色.

因为 oklch 是一个循环的颜色空间, 这里有一些获得所需颜色温暖的技巧.

- 暗模式: 非常冷的背景, 冷色调设置为 -20, 暖色调设置为 130.
- 亮模式: 背景有点暖, 设置 -70 为暖色调, 80 为冷色调.

如果你超出这个范围, 颜色可能会从冷变暖, 反之亦然.
> 您可以尝试各种选项值 ^_^

目标 CSS 变量: `var(--flexcyon-theme-warmth-offset)`

默认: 0

### 主题颜色亮度比

定义主题颜色的亮度比.

目标 CSS 变量: `var(--flexcyon-theme-brightness-ratio)`

默认: 1 (可变滑块, 范围从 0 到 2)

### 主题色饱和度

定义主题颜色的饱和度.

目标 CSS 变量: `var(--flexcyon-theme-saturation-ratio)`

默认: 1 (可变滑块, 范围从 0 到 5)

### 全局亮度比

目标 CSS 变量: `var(--flexcyon-brightness-ratio)`

默认: 1

### 全局对比度

目标 CSS 变量: `var(--flexcyon-contrast-ratio)`

默认: 1

> 例如, 如果你想像在 OLED 屏幕上获得更高的对比度, 可以尝试 1 到 1.5 之间的值

### 全局饱和比度

目标 CSS 变量: `var(--flexcyon-saturation-ratio)`

默认: 1

> 用于修改颜色的饱和度
