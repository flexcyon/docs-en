## 配置选项

### 选项卡编号

在 RTZ 模式下默认启用. 

目标 CSS 类: `.flexcyon-number-tabs`

默认: false (类切换)

### 删除选项卡标题

在 RTZ 模式下默认启用. 

目标 CSS 类: `.flexcyon-remove-tab-headers`

默认: false (类切换)

### 选项卡标题容器和选项卡内容的顺序相反

将选项卡标题容器放置在屏幕的底部而不是顶部, 对堆叠的选项卡没有任何作用.

目标 CSS 类: `.flexcyon-reverse-tab-header-container`

默认: false (类切换)
