---
title: 基础色
---

定义的颜色会覆盖背景和其他 UI 元素中使用的基础颜色的主题默认值. 

接受的格式: HEX

## 导航

```md
Style Settings
|-- flexcyon://Editor
|   |-- Colors
|   |   |-- Base Colors
```

## 配置选项

目标 CSS 变量: `var(--flexcyon-base-01)`
