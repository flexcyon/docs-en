## 配置选项

### 风格化针图标.

目标 CSS 类: `var(--flexcyon-stylised-pins)`

默认: true (类切换)

### Prevent accidental unpins

使用右键单击菜单或 Ctrl+W 取消绑定.

目标 CSS 类: `var(--flexcyon-prevent-accidental-unpin)`

默认: true (类切换)
