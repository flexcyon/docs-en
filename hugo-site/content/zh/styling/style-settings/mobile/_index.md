## 配置选项

### 还原 1.11 导航 UI

将 Vault 选择器和导航还原为 1.11 之前的样式.

目标类: `var(--flexcyon-revert-nav-item-alignment)`

默认: true (类切换)

### 在手机上显示状态栏

目标 CSS 变量: `var(--flexcyon-status-mobile-enabled)`

默认: false (类切换)
