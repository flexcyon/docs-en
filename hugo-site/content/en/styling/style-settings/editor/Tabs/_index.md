## Configuration Options

### Number tabs

Numbers your tabs in sequential order. Each tab group has its own counter. Meant for RTZ mode, but can be enabled separately.


CSS Classe(s) targeted: `.flexcyon-number-tabs`

Default: false (class toggle)

### Remove Tab Headers

Removes tab headers. You can still view the original content by hovering and a tooltip will appear. Meant for RTZ mode, but can be enabled separately.

CSS Classe(s) targeted: `.flexcyon-remove-tab-headers`

Default: false (class toggle)

### Reverse order of Tab Header Container and Tabs Content

Places tab header container at the bottom instead of the top of the screen, does not do anything for Stacked Tabs.

CSS Classe(s) targeted: `.flexcyon-reverse-tab-header-container`

Default: false (class toggle)
