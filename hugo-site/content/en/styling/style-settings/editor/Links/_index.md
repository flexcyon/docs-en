Accepted formats: HEX

## Configuration Options

### Internal Link Color

CSS Variable(s) targeted: `var(--flexcyon-link-color)`

Default (light mode):
<span class="col-sqr" style="background-color: #6f49ae"></span> #6f49ae


Default (dark mode): <span class="col-sqr" style="background-color: #d89ff5"></span> #d89ff5

### External Link Color
CSS Variable(s) targeted: `var(--flexcyon-external-link-color)`

Default (light mode): <span class="col-sqr" style="background-color: #3085dd"></span> #3085dd

Default (dark mode): <span class="col-sqr" style="background-color: #7ab8fe"></span> #7ab8fe

### Unresolved Link Color
CSS Variable(s) targeted: `var(--flexcyon-link-unresolved-color)`

Default (light mode): <span class="col-sqr" style="background-color: #6f49ae"></span> #6f49ae

Default (dark mode): <span class="col-sqr" style="background-color: #d89ff5"></span> #d89ff5

### Enable alternate link styling

Inspired by Willemstad theme's headings.

CSS Classe(s) targeted: `var(--flexcyon-alt-link-style)`

Default: false (class toggle)
