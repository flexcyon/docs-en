---
title: Workspace Components
---

Defines file line width when readable line length is enabled and other workspace
components like view header, titlebar and status bar.

Accepted Formats: x.y, rem

## Navigation

```md

|-- Flexcyon Style Settings

|   |   |-- Workspace
```

## Configuration Options

### File line width

CSS Variable(s) targeted: `var(--file-line-width)`

Default: 50 (rem)

### Editor Top Margin

CSS Variable(s) targeted: `var(--flexcyon-editor-margin-top)`

Default: 18 (px)

### Top Actions alignment

Alignment of the top actions (the ones with icons that allow you to make a
new note, new folder etc.)

CSS Variable(s) targeted: `var(--flexcyon-top-actions-alignment)`

Default: Center (variable select)

Options:
- Left
- Center
- Right
