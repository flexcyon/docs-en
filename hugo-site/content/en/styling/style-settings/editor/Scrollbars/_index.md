## Configuration Options

### Scrollbar width

CSS Variable(s) targeted: `var(--flexcyon-scrollbar-width)`

Default: 12 (px)

### Do not show scrollbar in settings

CSS Variable(s) targeted: `var(--flexcyon-settings-scrollbar-removed)`

Default: true (class toggle)
