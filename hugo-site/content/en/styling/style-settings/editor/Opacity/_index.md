## Configuration Options

### Opacity of dimmed elements

CSS Variable(s) targeted: `var(--dimmed)`

Default: 0.55

<span style="opacity: 0.55">Sample dimmed element</span>

### Opacity of Block labels
CSS Classe(s) targeted: `var(--flexcyon-block-label-opacity)`

Default: 0.55

<span style="opacity: 0.55">Sample dimmed element</span>

### Opacity of status bar in new tab

CSS Variable(s) targeted: `var(--flexcyon-status-new-tab-opacity)`

Default: 0.55

### Callout Metadata Background Opacity

CSS Variable(s) targeted: `var(--flexcyon-callout-bg-opacity)`

Default: 20%

<span style="opacity: 20%">Sample Background Opacity</span>

### Highlight Active Line Opacity

Opacity of highlight active line background.

CSS Variable(s) targeted: `var(--flexcyon-highlight-active-line-opacity)`

Default: 7.5%

### Animation initial opacity

CSS Variable(s) targeted: `var(--flexcyon-anim-start-opacity)`

Default: 0.55

### Alternate link styling opacity

CSS Variable(s) targeted: `var(--flexcyon-alt-link-opacity)`

Default: 55%

### Opacity of community items (unselected)

CSS Variable(s) targeted: `var(--flexcyon-comm-item-opacity)`

Default: 0.89
