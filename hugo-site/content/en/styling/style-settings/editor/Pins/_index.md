## Configuration Options

### Stylised pin icons

Stylise pin icons in the sidebars.

CSS Classe(s) targeted: `var(--flexcyon-stylised-pins)`

Default: true (class toggle)

### Prevent accidental unpins

Unpin with right click menu or Ctrl + W.

CSS Classe(s) targeted: `var(--flexcyon-prevent-accidental-unpin)`

Default: true (class toggle)
