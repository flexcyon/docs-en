Custom Callout / Callout Metadata:

- "title-`<opacity-value>`"

- "content-`<opacity-value>`"

- "all-`<opacity-value>`"

Options: Smaller, Small, Medium, Large

Configure in  [UI Font Size](../../Style-Settings/Editor/UI-Font-Size) settings

Usage:

```md
> [!info|title-smaller content-medium] The callout title will inherit the smaller UI Font size
> The callout content will inherit the medium UI Font size
```

```md
> [!info|all-small] The callout title will inherit the small UI Font size
> The callout content will inherit the small UI font size
```
