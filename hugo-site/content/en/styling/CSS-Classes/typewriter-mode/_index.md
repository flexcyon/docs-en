# Usage

Reduce opacity of all but the active line in source and live preview mode.
Opacity of inactive lines can be adjusted in Style Settings.

```md
cssclasses:
    - typewriter-mode
---

Lines before the active line are dimmed.

Active line has full opacity.

Lines after the active line are dimmed.
```
