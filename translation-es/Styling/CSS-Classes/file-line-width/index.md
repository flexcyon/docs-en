# Usage

Configurar el ancho de la línea de archivo en una base por nota

```md
cssclasses:
    - editor-50vw
---

Your content will have a file line width of 50 viewport width (50 percent of your
screen's size)
```

Other Values supported:

- `.editor-50vw`
- `.editor-55vw`
- `.editor-60vw`
- `.editor-65vw`
- `.editor-70vw`
- `.editor-75vw`
- `.editor-80vw`
- `.editor-85vw`
- `.editor-90vw`
- `.editor-95vw`
